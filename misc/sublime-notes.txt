Key Commands
--------------
command + n                         - New Untitled file
command + w                         - Close current file tab
command + #                         - Open tab #
command + tab                       - Switches between most recent files
command + p                         - Drop down of all open files, or inside files
command + up arrow                  - Move cursor to top of file
command + down arrow                - Move cursor to bottom of file
command + left arrow                - Move cursor to beginning of line
command + right arrow               - Move cursor to end of line
command + [ or ]                    - Change line indentation, highlight for multiple rows
command + x                         - Cut line and copy to clipboard
command + d                         - Select current word
command + l                         - Select current line
command + mouse click               - Create multi-line cursor
ctrl    + m                         - Move to closing brace
shift   + command + t               - Reopen last closed file
shift   + command + [ or ]          - Move to left/right file tab
shift   + command + v               - Paste and indent code
shift   + command + a               - Select all code between tags
shift   + command + j               - Select current indentation level
shift   + command + space           - Select current color item
shift   + command + p               - Command drop down menu
modal K + command + v               - Shows clipboard
modal K + command + up arrow        - Bring file into side column.
modal K + command + down arrow      - collapses side columns and rows
option + left/right arrow           - Move over one word at a time
shift + option + left/right arrow   - Select one word at a time

Plugins
--------
shift + command + c                 - Bring up color picker
